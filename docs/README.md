エンジニアとして4年間、Webアプリケーションのフロントエンドからバックエンドまで幅広く開発に携わってきました。
1社目では、急成長するWebサービスのフロントエンド開発を中心に担当し、業務効率化のための社内ツールの開発・運用を提案・実装しました。その結果、全社的に大幅な業務効率化を実現しました。
